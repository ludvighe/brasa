# Brasa /w TTCh

A bit intense but super christmasy fireplace in the terminal.

## Build

```sh
cargo build --release
```

## Use

```sh
./target/release/brasa
```

| Key | Command                      |
| --- | ---------------------------- |
| q   | Quit                         |
| i   | Toggle rendering information |
| s   | Toggle seconds in TTCh       |
